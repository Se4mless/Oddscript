# Oddscript language
- Oddscript is a language i quickly made one evening based off the language brainf#ck. 
- The Language Is Very Simple And I have setup a simple [[tutorial]]  for you to check out.
- Do Note that this language is not very well coded so if there are any bugs found, please post it as an issue.
## Installation:
- Make sure you have Cargo installed.
- In A Terminal run `cargo install --git https://github.com/Se4mless/Oddscript.git`
